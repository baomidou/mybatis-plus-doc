## Features

- **Non-invasion**ï¼šNo change but enhancement, no effect for current project by importing MP
- **Little loss**ï¼šInject CRUD sql automatically when startup, almost non-loss, OOP(Object-Oriented Programming)
- **Powerful CRUD**ï¼šgeneral Mapper/Service inside, you can CURD by few configuration, powerful condition constructor can simplify programming. for single table CRUD, XML is needless
- **Lambda**ï¼šSimplify query by using Lambda, and avoid column name typo
- **Multiple DB**ï¼šSupport MySQL, MariaDB, Oracle, DB2, H2, HSQL, SQLite, Postgre, SQLServer2005, SQLServer, etc.
- **Auto generate PK**ï¼š4 strategies for Primary Key, free configuration
- **ActiveRecord**ï¼šYou can use entity for CRUD, just let entity extends Model
- **Customize general methods**ï¼šYou can customize general methods, "Write once, use anywhere"
- **Code Generator**ï¼šYou can generate Mapper/Model(Entity)/Service/Controller, and customized code template is also supported(Freemarker, Velocity)
- **Pagination**ï¼šPhysical pagination, transparent to developers
- **Performance Interceptor**ï¼šCan show sql execution time (Suggest to open it when doing development)
- **Smart Interceptor**ï¼šCan stop execution of delete/update for full table, and customized intercepting rule is also supported, which can prevent misoperation
- **Sql Injection Interceptor**ï¼šCan detect and avoid the sql injection
