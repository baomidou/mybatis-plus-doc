Code Generator

AutoGenerator is the code generator of MyBatis-Plus. With AutoGenerator, you can quickly generate code for various modules such as Entity, Mapper, Mapper XML, Service, Controller, etc., greatly improving development efficiency.

Note

The old code generator is applicable to versions below 3.5.1. If you are using version 3.5.1 or above, please refer to the New Code Generator for configuration and usage. The new code generator is more concise and powerful, and it is recommended that everyone upgrade to it.

For more detailed configurations, please refer to the Code Generator Configuration article.
